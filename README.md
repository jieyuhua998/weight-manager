# 体重管理助手

一个简洁美观的体重管理前端应用，支持 BMI 计算和体脂率计算。

## 功能特点

- **BMI 计算器**: 根据身高和体重计算身体质量指数
- **体脂率计算器**: 使用美国海军体脂计算公式，根据性别、年龄、腰围、颈围等数据计算体脂率
- **美观的界面**: 现代化渐变设计，响应式布局
- **健康建议**: 提供实用的健康小贴士

## 如何使用

直接在浏览器中打开 `index.html` 文件即可使用。

## 技术栈

- HTML5
- CSS3 (渐变、动画、响应式设计)
- 原生 JavaScript
